# 🌿 Paradise Nursery

Paradise Nursery is a React-based web application that allows users to browse, select, and manage a variety of indoor and outdoor plants. The goal of this project is to provide an intuitive and visually appealing shopping experience for plant enthusiasts.

---

## 🏡 Project Overview

Paradise Nursery showcases a modern, responsive plant store built using **React** and **Redux Toolkit**.  
Users can:
- View a list of available plants with images, names, and prices  
- Add or remove plants from their shopping cart  
- Adjust quantities and view total costs  
- Navigate between pages using a clean, minimal interface  

---

## 🧩 Features

- **Landing Page:** Displays company name and a “Get Started” button  
- **About Us Page:** Describes the company’s mission and background  
- **Product Listing Page:** Displays plant categories and products  
- **Shopping Cart:** Allows users to manage selected items  
- **Responsive Design:** Works seamlessly across devices  

---

## 🛠️ Technologies Used

- **React.js** – Frontend framework  
- **Redux Toolkit** – State management  
- **React Router** – Navigation  
- **CSS3** – Styling and layout  
- **JavaScript (ES6+)** – Core logic
